# ai-toolkit

Repositorio para almacenar skills propios hechos por mí sobre las tecnologías que utilizo.

## Estructura

- `skills/`: carpeta principal para organizar skills por tecnología o tema.

## Uso recomendado

1. Crear una carpeta por tecnología dentro de `skills/`.
2. Guardar cada skill como archivo Markdown (`.md`) con un nombre descriptivo.
3. Documentar objetivo, contexto, pasos y ejemplos de uso.
