# Create Ebook

Step-by-step workflow for ebook creation.

## Prerequisites

- Topic validated (search demand, competition analyzed)
- Target audience defined
- Ebook type chosen (educational, reference, thought leadership)
- Word count target set (5K-15K for lead magnet, 30K-80K for full book)

## Phase 1: Planning

### Step 1: Research
- [ ] Use `prompts/ebooks/research-topic-for-ebook.md`
- [ ] Collect 10+ sources
- [ ] Identify 3 content gaps
- [ ] Document data points and stats

### Step 2: Title & Positioning
- [ ] Use `prompts/ebooks/generate-ebook-title-options.md`
- [ ] Select 3 title candidates
- [ ] Check domain/social handle availability (if brand-aligned)
- [ ] Finalize subtitle (if applicable)

### Step 3: Outline
- [ ] Use `prompts/ebooks/generate-ebook-outline.md`
- [ ] Define chapter count and word allocation
- [ ] Identify appendix content (glossary, resources)
- [ ] Review outline for flow and coverage

## Phase 2: Writing

### Step 4: Introduction
- [ ] Use `prompts/ebooks/write-ebook-introduction.md`
- [ ] Hook, promise, credibility, map, invitation

### Step 5: Draft Chapters
For each chapter:
- [ ] Use `prompts/ebooks/write-ebook-chapter.md`
- [ ] Include examples, code, diagrams
- [ ] Add callout boxes for insights
- [ ] Write chapter summary
- [ ] Mark as draft complete

### Step 6: Case Studies
- [ ] Use `prompts/ebooks/generate-ebook-case-study.md`
- [ ] 2-3 case studies minimum
- [ ] Include metrics and results

### Step 7: Exercises (if educational)
- [ ] Use `prompts/ebooks/generate-ebook-exercises.md`
- [ ] 3-5 exercises per chapter
- [ ] Include answer keys (in appendix)

### Step 8: Conclusion
- [ ] Use `prompts/ebooks/write-ebook-conclusion.md`
- [ ] Recap, action plan, final motivation, CTA

## Phase 3: Editing

### Step 9: Self-Edit
- [ ] Use `prompts/ebooks/edit-ebook-chapter.md` per chapter
- [ ] Check active voice, clarity, flow

### Step 10: Consistency Review
- [ ] Use `prompts/ebooks/review-ebook-consistency.md`
- [ ] Fix terminology, tone, formatting issues

### Step 11: Technical Review (if applicable)
- [ ] Run all code examples
- [ ] Verify version numbers
- [ ] Check URL validity

## Phase 4: Production

### Step 12: Formatting
- [ ] Use `prompts/ebooks/format-ebook-for-export.md`
- [ ] Apply typography standards
- [ ] Generate TOC
- [ ] Create front matter (title page, copyright, dedication)
- [ ] Export to PDF and EPUB

### Step 13: Cover Design
- [ ] Design cover (Canva/Figma)
- [ ] Verify title readability at thumbnail size
- [ ] Include author name and subtitle

## Phase 5: Publishing

### Step 14: Platform Setup
- [ ] Choose platform (Gumroad / Amazon KDP / Leanpub / own site)
- [ ] Upload files
- [ ] Set pricing (free / $9-$49 / premium)
- [ ] Configure landing page

### Step 15: Marketing
- [ ] Use `prompts/ebooks/generate-ebook-marketing-copy.md`
- [ ] Write landing page
- [ ] Prepare email sequence (3 emails)
- [ ] Create social media posts (5-10)
- [ ] Schedule launch announcement

## Phase 6: Post-Launch

### Step 16: Monitor & Iterate
- [ ] Track downloads/sales
- [ ] Collect reader feedback
- [ ] Update content (quarterly for technical ebooks)
- [ ] Plan follow-up content (next ebook, course, community)

## Verification Checklist

- [ ] All chapters written and edited
- [ ] Consistency review passed
- [ ] Code tested (if technical)
- [ ] Formatted for PDF and EPUB
- [ ] Cover designed
- [ ] Landing page live
- [ ] Marketing copy ready
- [ ] Email sequence scheduled
- [ ] Analytics tracking configured
