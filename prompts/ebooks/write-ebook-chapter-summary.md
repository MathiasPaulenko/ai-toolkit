# Write Ebook Chapter Summary

Write compelling summaries for each ebook chapter.

## Summary Types

### 1. Landing Page Preview (50 words)
- Hook + core promise
- Benefit-driven language
- "In this chapter, you'll discover..."

### 2. TOC Entry (1 sentence)
- What the chapter covers
- Why it matters

### 3. Back Cover / Sales Copy (30 words)
- Intrigue without spoilers
- Highlight unique angle

### 4. Social Shareable (1 tweet)
- One key insight
- Call to read more

## Writing Guidelines
- No spoilers for narrative content
- Quantify benefits where possible
- Use active, present-tense verbs
- Include the word "you" for engagement

## Output Format

```markdown
## Chapter Summaries

### Chapter 1: [Title]
**Landing page**: [50 words]
**TOC**: [1 sentence]
**Social**: [1 tweet]

### Chapter 2: [Title]
...
```
