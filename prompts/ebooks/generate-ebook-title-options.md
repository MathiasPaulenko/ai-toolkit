# Generate Ebook Title Options

Given a topic and audience, generate title options using proven formulas.

## Title Formulas

### 1. How-To / Instructional
- "How to [Achieve Result] Without [Common Obstacle]"
- "The [Audience] Guide to [Outcome]"

### 2. List / Curiosity
- "[Number] [Adjective] Ways to [Outcome]"
- "The [Number] [Noun] Every [Audience] Should Know"

### 3. Transformation / Journey
- "From [Before] to [After]: A [Audience] Journey"
- "Mastering [Topic]: The [Audience] Blueprint"

### 4. Problem / Solution
- "Why [Common Problem] Happens (and How to Fix It)"
- "The Real Reason [Audience] Struggle With [Topic]"

### 5. Authority / Definitive
- "The Definitive Guide to [Topic]"
- "[Topic]: A [Year] [Audience] Reference"
